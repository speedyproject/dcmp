# DCMP

&#12288;&#12288;该项目基于Consul作为注册发现，服务端以SpringBoot实现，无代码侵入模式编程，支持原生web应用(Servlet) <br/>
,Struts2,SpringProject。"0"配置 (无任何XML) 使用简单方便，配置自动更新，Git仓库配置文件一旦更改配置数据也 <br/>
即刻刷新，不需要MQ。低侵入性或无侵入性、强兼容性。<br/>
<br/>
